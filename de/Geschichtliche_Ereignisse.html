Geschichtliche Ereignisse zu Ihrem Stammbaum hinzufügen

webtrees ermöglicht Ihnen, Geschichtsdaten einzufügen, um das Leben Ihrer Vorfahren in einem historischen Kontext besser darstellen zu können. Diese Ereignis-Chronologie wird durch die Erstellung einer Liste von Daten ermöglicht, mit denen webtrees die Geschichtsdaten auf der Registerkarte "Fakten und Ereignisse" auf jeder Personenseite anzeigen kann. Diese Funktion gibt es nur auf der Personenseite.

Konfiguration Ihrer Webseite zur Darstellung geschichtlicher Ereignisse

Das Hinzufügen der Funktion für geschichtliche Ereignisse in webtrees ist einfach. Erstellen Sie eine Datei mit Ereignissen und Daten und installieren Sie sie im Ordner /data Ihrer webtrees Webseite. Die Datei sollte gemäß php Standard gebaut und in der Weise benannt sein, dass jeder Satz von Ereignissen einer Sprache zuzuordnet werden kann: "histo.xx.php" (z.B.: histo.de.php für Deutsch oder histo.en_us.php für US-Englisch). Siehe unten. Ersetzen Sie das Sprachkürzel xx mit dem Code der anderen Sprache, um Ereignisse in einem anderen Land einzufügen. Die Installation der histo.xx.php Datei führt dazu, dass webtrees diese Option automatisch anzeigt. Besucher auf Ihrer Webseite können dann einfach durch Aktivieren des entsprechenden Kontrollkästchens, die "Geschichtlichen Ereignisse" anzeigen, die sich auf die Lebenszeit der jeweiligen Person beziehen.

Datei histo.xx.php erstellen

Eine solche Datei ist eine PHP-Datei, die ein paar einfache Syntax-Regeln beachten sollte:

  • Beginnen Sie mit einer Kopfzeile, beachten Sie die php Syntax-Regeln bei der Eingabe und schließen Sie dann mit einer Fusszeile.
  • Es ist nicht erforderlich, PHP-Programmierung zu verstehen, um diese Datei zu erstellen.
  • Verwenden Sie einen Text-Editor, der das Speichern von Dateien ohne BOM (Byte Order Mark) ermöglicht. Notepad++ ist ein Open Source Programm (GPL-Lizenz) und kann heruntergeladen werden unter DownloadNotepad++, so wie TextWrangler für Mac-Nutzer.

Datei Kopfzeile

Kopieren Sie folgenden Code und fügen ihn ein: 

<?php
// File: /data/histo.xx.php (where xx is a language code such as "fr" or "en_GB")
// Display historical facts on the personal facts tab of the individual page 
// Each line is a GEDCOM record to describe an event.
// Events must contain dates, and will only be shown if the individual was alive on that date.

if (!defined('WT_WEBTREES')) {
   header('HTTP/1.0 403 Forbidden');
   exit;
}

Datei Inhalt - die Liste der Ereignisse

Hier ist ein Beispiel:

Fügen Sie die folgende Zeile ein (jede Zeile gehört zu einem Ereignis): 

 $histo[] = "1 EVEN\n2 TYPE Bau der Berliner Mauer\n2 DATE 13 AUG 1961\n2 NOTE symbolischer \"Tag des Mauerbaus\";, 
 s. http://de.wikipedia.org/wiki/Berliner_Mauer";

wird dargestellt als:

Hinweis: Dieses Beispiel zeigt, dass es auch möglich ist, einen Hyperlink hinzuzufügen. Er wird korrekt von webtrees interpretiert.

Was Sie ändern können, wird hier rot dargestellt: 

$histo[] = "1 EVEN\n2 TYPE Titel\n2 DATE DD MMM YYYY\n2 NOTE Inhalt der Ereigniszeile";
  • Titel: ist die Kurzbezeichnung des Ereignisses; schreiben Sie z.B. Beginn Zweiter Weltkrieg.
  • DD MMM YYYY: ist das Datum, DD für Tag mit ein oder zwei Ziffern, MMM für Monat mit drei Buchstaben und YYYY für das Jahr mit vier Ziffern; schreiben Sie z.B. 01 SEP 1939.
  • Inhalt der Ereigniszeile: schreiben Sie erläuternden Text; z.B. durch den deutschen Überfall auf Polen.

Sie würden die Zeile erhalten: 

$histo[] = "1 EVEN\n2 TYPE Beginn Zweiter Weltkrieg\n2 DATE 01 SEP 1939\n2 NOTE durch den deutschen Überfall auf Polen"


Speichern Sie die Datei.

erstellte Datei verwenden

Jetzt ist alles, was Sie tun müssen, die Datei in den Ordner /data von webtrees zu kopieren und dann Ihren Stammbaum zu besuchen. Wählen Sie eine Personenseite und aktivieren Sie das Kontrollkästchen "Geschichtliche Ereignisse". Sie werden die Ereignisse innerhalb der Daten der jeweiligen Person eingefügt sehen.

Hinweis: Um Buchstaben mit Akzenten korrekt anzuzeigen, sollte die Datei mit UTF-8-Kodierung gespeichert werden. Aus historischen Gründen akzeptiert webtrees HTML-Instanzen wie &agrave; für à, aber dies wird nicht empfohlen.

Einige Möglichkeiten für Ereignisse

Es könnte interessant sein, folgende Geschichtliche Ereignisse einzufügen:

  • bedeutende wissenschaftliche Ereignisse für eine Familie von Wissenschaftlern,
  • die großen literarischen Ereignisse für eine Familie von Schriftstellern,
  • wichtige Daten der Kunstgeschichte für eine Familie von Malern und Künstlern,
  • etc...
