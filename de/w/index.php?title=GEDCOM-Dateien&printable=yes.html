GEDCOM-Dateien

Aus webtrees
Wechseln zu: Navigation, Suche

Wenn Sie einen Stammbaum (GEDCOM-Datei) zu webtrees HINZUFÜGEN, IMPORTIEREN oder HOCHLADEN werden alle Daten aus der Datei in die Datenbanktabellen übertragen. Die Datei selbst wird nicht mehr von webtrees verwendet oder benötigt.

  • Wenn Sie HINZUFÜGEN oder IMPORTIEREN verwenden, bleibt die Datei im webtrees/data Ordner, in den Sie sie zuerst kopiert haben und wird nicht durch nachträgliche Bearbeitung der webtrees Daten geändert.
  • Wenn Sie HOCHLADEN verwenden, verbleibt die Datei an ihrem Ursprungsort und bleibt ebenfalls unberührt.

Wenn Sie Ihre genealogischen Daten außerhalb von webtrees ändern, ist es nicht notwendig, Ihre GEDCOM-Datei oder Datenbank aus webtrees zu löschen und von vorne zu beginnen. Befolgen Sie diese Schritte, um eine GEDCOM zu aktualisieren, die bereits importiert wurde:

  1. Entscheiden Sie, ob Sie Ihren neuen Stammbaum (GEDCOM-Datei) hochladen oder importieren wollen.
    • Verwenden Sie "Eine Datei auf Ihrem Computer", wenn Ihre Stammbaum-Datei auf ihrem lokalen Rechner liegt und kleiner ist als die maximal zulässige PHP Datei-Ladegröße Ihres Servers (manchmal nur 2 MB). Die neue Datei kann jeden beliebigen Namen haben.
    • Verwenden Sie "Eine Datei auf dem Server" für größere Dateien. In diesem Fall müssen Sie ein FTP-Programm verwenden, um die Datei zuerst von Ihrem lokalen Rechner in den webtrees/data-Ordner auf Ihrem Server zu transferieren. Entweder überschreiben Sie eine eventuell dort bereits vorhandene Datei oder verwenden einen anderen Namen.
  2. Auf der Seite Verwaltung wählen Sie die webtreesStammbaum (GEDCOM)-Konfigurationsseite, indem Sie auf "Stammbäume" klicken. Dann wählen Sie Ihren gewünschten Stammbaum (GEDCOM-Datei) oder wählen "Einen neuen Stammbaum erstellen".
  1. Bitte beachten Sie die Option für Medien-Objekte ("Wenn Sie die Medienobjekte in webtrees erstellt haben und Sie Ihre GEDCOM-Datei mit Hilfe einer anderen Software auf Ihrem Computer erstellt oder bearbeitet haben, die Medienobjekte löscht, dann aktivieren Sie dieses Feld, damit die aktuellen Medienobjekte mit der neuen GEDCOM verknüpft werden."). In den meisten Fällen sollten Sie diese Option DEAKTIVIERT lassen.
  2. Klicken Sie auf "SPEICHERN". webtrees wird die GEDCOM-Datei vor dem Import erneut überprüft. Während dieses Prozesses kopiert webtrees Ihren gesamten Stammbaum (GEDCOM-Datei) in eine Dummy-Tabelle in Ihrer Datenbank. Je nach Kodierung Ihrer Datei, der Dateigröße, den Fähigkeiten Ihres Servers und der unterstützenden Software kann dies einige Zeit dauern. Kein Fortschrittsbalken wird angezeigt, während die Daten kopiert werden, und sollten Sie von dieser Seite weg navigieren, wird der Prozess zeitweilig unterbrochen. Er wird wieder gestartet, wenn Sie zur Stammbaum-Management-Seite zurückkehren.

FORMAT

Jedes Programm für Familiengeschichte hat seine eigene Methode GEDCOM-Dateien zu erstellen und unterschiedliche Optionen zur Auswahl des Ausgabeformats. webtrees Import Routinen können viele verschiedene Formate lesen, aber nicht unbedingt alle. Wenn Ihre Software eine "UTF8"-Option hat, sollten Sie sie immer verwenden. Allerdings wurde webtrees mit folgenden, alternativen Formaten getestet:

  • ANSI - Import OK, ist aber langsam aufgrund der Übersetzung in UTF8 als Teil des Import-Prozesses.
  • MAC - Import OK, ist aber langsam aufgrund der Übersetzung in UTF8 als Teil des Import-Prozesses.
  • DOS - Import OK, ist aber langsam aufgrund der Übersetzung in UTF8 als Teil des Import-Prozesses.
  • ANSEL - Import derzeit nicht möglich. Zeigt die Warnung an "Fehler: GEDCOM-Datei kann nicht aus ANSEL zu UTF8-Codierung konvertiert werden". Spätere Versionen von webtrees können möglicherweise eine Übersetzung von ANSEL zu UTF8 enthalten, aber das ist kein einfacher Prozess.
