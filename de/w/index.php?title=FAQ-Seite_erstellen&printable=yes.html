FAQ-Seite erstellen

Aus webtrees
Wechseln zu: Navigation, Suche

Dies ist eine Kurzanleitung, die beschreibt, wie man eine Seite für „Häufig gestellte Fragen“ erstellt. Das webtrees Support-Team empfiehlt dringend, das Erstellen einer Seite für „Häufig gestellte Fragen“ als einen der wichtigsten Teile für eine ordnungsgemäß konfigurierte webtrees Seite zu erwägen.


Der Link auf diese Seite, wird, sobald er aktiviert ist, im Hilfe-Menü erscheinen. Diese Seite sollte sich natürlich inhaltlich auf Ihre Webseite beziehen: Sie können Hintergrund-Infos über Ihre Seite anzeigen, Hinweise geben, wie die Website zu benutzen ist, oder vielleicht Links auflisten, mit denen Benutzer zusätzliche Informationen oder Hilfe finden können. Vielleicht kann Sie auch die allgemeinen Nutzungsbedingungen und Ihre Datenschutzerklärung enthalten.

In webtrees können nur Administratoren oder Manager Seiten für Häufige Fragen (FAQ) erstellen und verwalten; nur Administratoren verfügen jedoch über die Berechtigung zur Erstellung der ersten Seite für Häufige Fragen (FAQ); danach können auch Manager, im jeweiligen Stammbaum, den sie verwalten, Seiten hinzuzufügen, bearbeiten oder löschen.

Um Ihre Seite für Häufige Fragen (FAQ) zu initialisieren und zu erstellen:

  1. Navigieren Sie zu Ihrer Seite Verwaltung.
  2. Klicken Sie in der Menü-Liste im linken Bereich auf das Wort Module.
  3. Navigieren Sie zu dem Modul Häufige Fragen (FAQ), stellen Sie sicher, dass Sie die Funktion Aktiviert haben und klicken Sie dann am unteren Rand der Seite auf Speichern.
  4. Klicken Sie in der Liste im linken Bereich auf Häufige Fragen (FAQ) (am unteren Ende der Menüleiste).
  5. Klicken Sie auf FAQ Eintrag hinzufügen.
  6. Geben Sie eine Frage (Titel) und Antwort ein.
  7. Speichern Sie Ihre Eingabe.

Hinweis: Wenn Sie Ihre neu erstellte Seite zu jeder Sprache hinzufügen möchten, wählen Sie keine Sprachen aus. Ihre Seite wird für die Benutzer in jeder Sprache sichtbar sein. Wählen Sie eine bestimmte Sprache nur aus, wenn Sie die Häufigen Fragen (FAQ) für eine bestimmte Sprache erstellen und eingeben wollen.

Nachdem Sie eine Seite für Häufige Fragen (FAQ) erstellt haben, können Sie auf die Hauptseite zurückkehren. Sie sehen den Link auf die neue Seite im Hilfe-Menü. Mit einem Klick auf dieses Symbol wird die Liste Ihrer Häufigen Fragen (FAQ) erscheinen (zu Beginn nur der neue Eintrag). Wenn Sie als Manager oder Administrator angemeldet sind, können Sie jetzt weitere Einträge hinzuzufügen oder bearbeiten, ohne auf die Seite Verwaltung zurückzukehren.
