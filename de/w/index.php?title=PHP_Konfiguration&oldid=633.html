Zu den Anforderungen für webtrees gehört die folgende Aussage über PHP:

  • PHP 5.3 oder höher. Falls Ihr Server nur PHP 5.2 bietet, müssen Sie die letzte webtrees Version 1.4.6 installieren. Beachten Sie, dass viele Internet-Provider sowohl PHP4 als auch PHP5 dabei oft mit PHP4 als Voreinstellung anbieten. In diesem Fall können Sie bestimmt mit Hilfe eines Bedienfelds oder einer Konfigurationsdatei zwischen beiden wechseln. Beachten Sie die Hilfe-Dokumentation Ihres Internet-Providers für weitere Informationen.
    • PHP sollte mit der PHP/PDO-Bibliothek für MySQL konfiguriert sein. Dies ist eine Option zur Server-Konfiguration. Sie ist standardmäßig auf den meisten Servern aktiviert. Siehe http://php.net/pdo
    • PHP sollte so konfiguriert sein, dass ausreichende Server-Ressourcen (Speicher und Ausführungszeit) für die Größe des Systems bereitgestellt werden. Typische Anforderungen sind:
      • Kleine Systeme (500 Personen): 16-32 MB, 10-20 Sekunden
      • Mittlere Systeme (5000 Personen): 32-64 MB, 20-40 Sekunden
      • Große Systeme (50000 Personen): 64-128 MB, 40-80 Sekunden


Der Prozess, um diese und weitere Konfigurationsoptionen einstellen zu können, kann variieren je nach Ihrem Internet-Provider. Der einfachste Weg, sie zu konfigurieren, ist die Bearbeitung in der System php.ini-Datei entweder durch Sie selber oder Ihren Internet-Provider. Dies sollte die ERSTE Lösung sein, die Sie versuchen.

Leider erlauben einige Provider diese Option nicht. Eine Alternative, die von einigen angeboten wird, ist für Sie eine .htaccess-Datei mit den Konfigurationseinstellungen zu erstellen. Diese wird dann mit Ihrer webtrees Installation (genaue Lage ist vom Server abhängig) gespeichert. Im Folgenden sind einige Beispiele dafür.

(WICHTIGER HINWEIS - diese gelten oft nur für bestimmte Internet-Provider. Ihrer kann ähnlich, aber nicht identisch sein. Prüfen Sie stets die genauen Details mit Ihrem Internet-Provider.)

PHP5 aktivieren

Concentric.com erklärt: "Wir unterstützen derzeit PHP Version 5.2.3, 4.4.7 und 3. Der Standard ist php4. Der einzige Weg, php5 zu verwenden, ist mit der htaccess-Anweisung. Die htaccess-Anweisung arbeitet auch mit php3. Für php5 ergänzen Sie CGIMapExtension /usr/local/bin/php5 php.

Datum-Zeitzone

Lunarpages erklärt: "Erstellen oder Bearbeiten von .htaccess im Webserver-Wurzel-Ordner mit einer Zeile SetEnv TZ' location'. 'location' kann gefunden werden in http://www.php.net/manual/en/timezones.php."

Ein komplettes Beispiel wäre: SetEnv TZ America/Chicago
