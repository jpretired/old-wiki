In webtrees gibt es quasi zwei "Portalseiten" für unterschiedliche Zwecke, auf die auch nur unter ihrem jeweiligen Namen Bezug genommen werden sollte:

  • Die erste ist die Startseite. Das ist die Seite, die die meisten Besucher sehen, wenn sie Ihre Webseite aufrufen (es sei denn, in Ihrer Konfiguration ist unter Verwaltung -> MeinStammbaum -> Datenschutz "Besucher Login notwendig" auf "Ja" eingestellt). Der Inhalt dieser Seite kann für jeden Stammbaum auf Ihrer Webseite variiert werden.
  • Die zweite wird als Mein webtrees bezeichnet und ist eine individuelle Seite für jeden Ihrer registrierten Benutzer. Wenn ein Benutzer sich auf Ihrer Webseite von der "Startseite" aus anmeldet, wird er auf seine eigene "Mein webtrees" Seite geleitet. (Hinweis: wenn er sich von einem anderen Teil der Website aus anmeldet, wird er auf die Seite, von der aus "Login" ausgewählt wurde, zurückgeleitet, sobald er vom System authentifiziert wurde). Diese Seite bleibt die gleiche für jeden Benutzer und für jeden Stammbaum, zu der er auf Ihrer Webseite Zugang hat.

Beide Seiten bestehen aus einer Reihe von "Blöcken" zur Anzeige verschiedener Informationen über Ihren Stammbaum. Diese Blöcke können auf der Seite (nach oben, unten, links und rechts) verschoben oder gelöscht werden. Es gibt verschiedene Blöcke, die zu beiden Seiten hinzugefügt werden können. Es gibt aber auch einige Blöcke, die nur für die eine oder andere Seite gedacht sind. Ein solches Beispiel sind die "Favoriten", die auf beiden Seiten zwar identisch heißen, aber völlig unterschiedliche Inhalte haben können.

Verwaltung - Standardblöcke einstellen

"Startseiten" Blöcke konfigurieren

Es gibt zwei Stellen, an denen "Startseiten"-Blöcke konfiguriert werden können, aber in beiden Fällen können nur registrierte Benutzer mit der Rolle "Verwalter" oder "Administrator" sie ändern.

  • Auf den Verwaltungseiten können Startseiten Blöcke voreingestellt werden, die von jedem auf Ihrer Webseite neu hinzugefügten Stammbaum benutzt werden (Verwaltung -> Stammbäume -> Standardblöcke anordnen). Diese "Standard"-Einstellung gilt nur für Bäume, die hinzugefügt werden, NACHDEM der Standard festgelegt wurde. Nach der Erstinstallation von webtrees ist die Einstellung so gewählt, dass Blöcke angezeigt werden, die die Entwickler durch Erfahrung gefunden haben und die am häufigsten verwendet werden.
  • Auf der Startseite selbst, ist am unteren Seitenrand ein Link mit dem Titel "Blöcke auf der Seite ändern". Damit können Sie die Blöcke für den spezifischen Stammbaum, von dem die Startseite angezeigt wird, ändern.

"Mein webtrees" Blöcke konfigurieren

Hier gibt es auch zwei Stellen, an denen diese Blöcke konfiguriert werden können. In diesem Fall ist einer nur für Administratoren verfügbar. Das geschieht auf der Seite Verwaltung -> Benutzer -> Standardblöcke anordnen. Diese "Standard"-Einstellung gilt nur für Benutzer, die neu hinzugefügt werden, NACHDEM der Standard festgelegt wurde.

Jeder Benutzer hat dann die Möglichkeit, die Blöcke auf seiner eigenen "Meine webtrees" Seite zu ändern. Der Link hierfür befindet sich am unteren Seitenrand mit dem Titel "Blöcke auf der Seite ändern" (s. Beispiel oben).

Unabhängig davon für welche Seite Sie die Blöcke konfigurieren oder welchen Link Sie verwenden, um die Einstellungen zu bearbeiten, die Benutzer-Oberfläche für die Änderung dieser Blöcke wird immer ähnlich aussehen, wie im Bild unten, nur in der Farbe unterschiedlich je nach verwendetem Thema Ihrer Webseite.

Die verfügbaren Optionen sind unkompliziert und erfordern nur zwei Operationen:

  1. Klicken Sie auf den Blocknamen, den Sie hinzufügen, entfernen oder verschieben möchten.
  2. Klicken Sie auf die entsprechenden Pfeile, um den Block nach links, rechts, oben oder unten zu verschieben. Die Doppelpfeile nach links und rechts bewegen die Blöcke vom linken zum rechten Bereich oder umgekehrt. Einzelpfeile nach links oder rechts bewegen die Blöcke zu oder von der Liste "Verfügbare Blöcke", so dass sie in einem der beiden Anzeigebereiche hinzugefügt oder entfernt werden.
