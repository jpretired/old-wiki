Quellen, Zitate und Archive

Aus webtrees
Wechseln zu: Navigation, Suche


Es wird dringend empfohlen, dass Sie für alle Informationen, die Sie in Ihre Datenbank eingegeben, eine Quelle erfassen. Sie werden vielleicht denken, dass es selbstverständlich oder trivial ist, aber andere Benutzer nicht, und sollten zu einem späteren Zeitpunkt widersprüchliche Informationen erkannt werden, werden Sie bereuen, es unterlassen zu haben.

Quellen sind Teil einer Hierarchie von Referenzen mit Zitaten und Archiven auf beiden Seiten.

  • Eine Quelle ist, wo Ihre Informationen her kommen. Das kann alles sein wie z.B. ein Eintrag in einem Kirchenbuch, ein Foto von einem Grabstein, etwas, das Sie an Tante Sally erinnert, als sie es bei einem Familienessen gesagt hat und Sie zu fünft waren oder eine Webseite.
  • Ein Zitat ist der eigentliche Inhalt der Quelle, aus der Sie Ihre Informationen beziehen wie z.B. eine Seitenzahl in einem Buch, eine Zeile in einem Kirchenbuch oder die Platzierung auf einer Webseite.
  • Ein Archiv ist eine Sammlung von Quellen, wie eine Urkundensammlung oder ein Friedhof. Das ist der Ort an dem das Kirchenbuch, der Grabstein oder das Photo abgelegt sind. Es könnte auch der Eigentümer der Webseite sein.

Eingabe von Quellenangaben

In jedem Bearbeitungsfenster für "Tatsachen und Ereignisse" gibt es die Option für "Neues Quellenzitat hinzufügen".

Alle Quellenangaben müssen von einer bestimmten Quelle kommen. Wenn Sie die gleiche Quelle bereits verwendet haben, können Sie den Anfang des Namens in das Textfeld eingeben und aus der erscheinenden Auswahl-Liste wählen (sofern Autovervollständigung aktiviert ist). Sie können auch auf die Schaltfläche "Finde eine Quelle" neben dem Textfeld klicken, um nach der Quelle zu suchen.

Wenn die Verbindung zur Quelle hergestellt wurde, können Sie weitere Informationen in den übrigen Feldern eingeben:

  • Zitat Einzelheiten: Wie das Zitat in der Quelle zu finden ist. Wenn Sie aus einem Buch zitieren, wird das die Seiten- oder Blatt-Zahl sein, aber andere Arten von Quellen können unterschiedliche Formate erfordern ("Auf der linken Seite der Buche vor der Kirche", "http://sourcesite.com/sourcepage.html", etc.)
  • Text: Der zitierte Text selbst.
  • Medien Objekt: Sie können zum Beispiel die Aufnahme einer Seite aus einem Kirchenregister oder einer anderen Quelle hinzufügen. Das ist nicht nur ein guter Weg, um Ihre Website abwechslungsreich und interessant zu gestalten, es ist auch eine Zeitersparnis, falls Sie die Quelle später erneut aufsuchen müssen, und ein Schutz, für den Fall, dass die Webseite, von der Ihre Aufnahme stammt, abgeschaltet werden sollte. (Achtung: Falls Sie fremde Bilder verwenden, sollten Sie sich vor Urheberrechtsthemen hüten; es liegt in Ihrer Verantwortung, sicherzustellen, die Rechte zum Rreproduzieren von Bildern zu haben).
  • Bemerkung: Jeder Text, den Sie in das Textfeld eingeben, wird in dem Datensatz der Person, zu der die Quelle gehört, gespeichert und ist somit an anderer Stelle nicht verfügbar. Wenn Sie den Text in anderen Aufzeichnungen wieder verwenden wollen, sollten Sie ihn stattdessen als "gemeinsame Bemerkung" eingeben.

Achtung: Wenn Sie keine Quelle angeben, gehen die Informationen, die Sie in den anderen Feldern eingeben, verloren!

Erstellen von neuen Quellen

Wenn Sie eine Quelle zum ersten Mal verwenden, benutzen Sie die Schaltfläche "Neue Quelle erstellen".

Geben Sie die Details zur Quelle in dem erscheinenden Fenster ein. Der Titel ist erforderlich; die anderen Felder sind optional.

  • Abkürzung: Dies ist ein optionales Feld zur Eingabe einer Kurzform des Titels. Beachten Sie, dass in einigen genealogischen Programmen die Abkürzung erforderlich ist.
  • Verfasser und Veröffentlichung: Diese Felder sind nützlich, wenn Sie aus einem Buch zitieren. Für andere Arten von Quellen können sie fallweise von Nutzen sein.

Achtung: Das Fenster Quelle ist nicht der Platz, wo die Zitate der Quelle eingetragen werden.

Wenn Sie alle Felder, die Sie verwenden möchten, ausgefüllt haben, klicken Sie auf "Speichern".

Bei der Erstellung der Quelle aus einem Zitat-Fenster, erscheint eine Meldung wie "Fügen Sie die folgende ID in Ihr Bearbeitungs-Feld ein, um zum neu erstellten Datensatz zu verknüpfen". Diese Nachricht ist ein Link und durch Klicken wird die Aktion ausgeführt. Wenn Sie nur das Fenster schließen, wird die Quell-ID nicht im Zitat übernommen. Die Quelle geht nicht verloren, aber Sie müssen die Verbindung dann manuell herstellen.

Archive

Eine Quelle kann auch zu einer größeren Sammlung von Quellen gehören. Das Verfahren, um eine Quelle mit einem Archiv zu verbinden ist vergleichbar zu der Art wie man Zitate zu Quellen hinzufügt.

Die Signatur ist der Code, mit dem die Quelle im Archiv identifiziert wird.

Zuverlässigkeit

Eine Quelle ist nicht die Wahrheit - es ist nur ein Beweisstück zur Stützung von Schlussfolgerungen, die Sie ziehen, zum Beispiel, dass jemand am 19. Dezember 1769 geboren wurde.

Nicht alle Quellen sind gleichermaßen zuverlässig oder glaubwürdig. In webtrees können Sie Ihre eigene Beurteilung für die Zuverlässigkeit einer Quelle eingeben. Dies wird im Feld "Daten-Qualität" vorgenommen und erfolgt über eine Auswahlliste. Es ist üblich, eine Nummerierung für die Qualitätsbewertung zu verwenden:

3 = Direkter Beweis (primäre Beweismittel, unwiderlegbare Beweislage)
2 = Indirekter Beweis (sekundäre Beweismittel, Daten irgendwann nach dem Ereignis offiziell aufgezeichnet)
1 = Fraglicher Beleg (fragwürdige Beweismittel, Interviews, Volkszählung, mündliche Familienüberlieferung, potenzielle Verfälschung, z.B. Autobiographie)
0 = Unsicherer Hinweis (Unzuverlässige Beweismittel, geschätzte Daten)

Das Feld "Datenqualität" ist in der Standard-Installation von webtrees nicht sichtbar. Es gilt als eine erweiterte Funktion, die vom Administrator unter Verwaltung -> mein Stammbaum -> Optionen bearbeiten -> "Vollständige Quellen-Zitate benutzen" aktiviert werden muss.

Online-Quellen

Heutzutage weisen viele Quellenangaben auf Online-Quellen: Scans von Kirchenbüchern, Online-Datenbanken, andere genealogische Webseiten, etc.. Die GEDCOM-Spezifikation wurde nicht unter Berücksichtigung dieser Situation geschrieben, aber webtrees bietet gute Werkzeuge, um solche Quellen auch zu behandeln.

Wenn Ihr Zitat von einer Web-Seite stammt, geben Sie die Internetadresse im Feld "Zitat Details" ein. Dies ist ein Freitextfeld, so dass Sie auch zusätzliche Informationen eingeben können, wie die Seitenzahl eines gescannten Buches. Es ist eine gute Idee, auch das Datum, an dem die Seite abgerufen wurde, einzugeben, da sich Web-Adressen ändern können.

Jede Internetadresse, die Sie in dieses Feld eingeben, wird automatisch in einen anklickbaren Link umgewandelt.

Wenn die Seite, von der Sie zitieren, ein Teil einer größeren Website ist, deren Adresse Sie auch aufnehmen möchten, gibt es hierfür verschiedene Wege.

  • Wenn die Web-Seite Teil einer noch größeren Seite ist (z.B. Scans aus einem bestimmten Buch in einem "virtuellen Archiv"), können Sie die Haupt-Seite zum Archiv und die Adresse der Zwischenseite zur "Signatur" machen. Das wäre der Fall, wenn eine Sammlung seine Archive online veröffentlicht: Die Haupt-Webseite der Sammlung wäre das Archiv; die Portal-Seite für das Buch, aus dem Sie zitieren, wäre die Quelle; die Internetadresse würde eingegeben in der Signatur; und die Internetadresse der Seite, von der Sie zitieren, würde im Feld Zitat Details eingegeben werden.
  • Wenn die Webseite nicht Teil einer größeren Struktur ist, können Sie immer noch das Archiv--Quelle--Zitat Modell verwenden.
  • Wenn Sie die Web-Seite nicht als Archiv bezeichnen möchten, gibt es andere Felder, in denen Sie die Internetadresse eingegeben können. Wenn Sie zur Hauptseite der Quelle navigieren, gibt es ein Auswahlmenu von Fakten. Abhängig von den Einstellungen der Webseite, kann ein WWW Feld verfügbar sein. Ansonsten können Sie die Adresse in einer Bemerkung oder unter Veröffentlichungs-Details eingeben. Derzeit wird jede Internetadresse in einer Bemerkung in einen anklickbaren Link umgewandelt, aber das geschieht nicht in einem Veröffentlichungs-Feld.
    Achtung: Die Benutzung der WWW-Angabe für eine Quell-Adresse erzeugt GEDCOM Code, der sich nicht streng an die Norm hält. Sie müssen sich entscheiden, ob dies ein Problem darstellt oder nicht.
