Installieren von webtrees dauert nur wenige Minuten. Alles, was Sie brauchen, ist ein Webserver mit PHP und MySQL. Fast jeder Web-Hosting-Service bietet das.

Wenn Sie Kommando-Zeilen Zugang auf Ihrem Server und Administrator-Rechte haben, können Sie webtrees auch unter Verwendung eines RPM Paketverwaltungs-Systems installieren.

Laden und installieren Sie die Applikation

  • Laden Sie die letzte Version von webtrees von http://launchpad.net/webtrees.
  • Entpacken Sie die Dateien und laden Sie sie in einen leeren Ordner auf Ihrem Webserver.
  • Öffnen Sie Ihren Web-Browser und geben Sie die Adresse für Ihre webtrees Webseite (z.B. http://www.meinserver.com/webtrees) in die Adresszeile ein.
  • Der webtrees Setup-Assistent wird automatisch gestartet.

Server Konfigurations-Prüfung

Der Installations-Prozess beginnt mit einer Seite zur Bestätigung der Eignung Ihres Servers. Wenn es irgendwelche Probleme gibt, werden sie hier aufgelistet.


Datenbank Verbindung

webtrees benötigt eine Datenbank. Die Verbindungsparameter werden von Ihrem Server-Administrator zur Verfügung gestellt.


Datenbank Konfiguration

Sie müssen webtrees mitteilen, welche Datenbank verwendet werden soll.

Wenn Sie den Namen der Datenbank nicht wissen, fragen Sie Ihren Server-Administrator.

Wenn sich webtrees die Datenbank mit anderen Anwendungen (oder anderen webtrees Installationen) teilt, können Sie für alle Tabellen ein eindeutiges Präfix wählen.


Administrator-Konto und email-Einstellungen

Sie müssen ein Administrator-Konto erstellen. Sie haben auch die Möglichkeit, Ihre email zu konfigurieren.

Nach einem Klick auf continue, kann es zu einer Verzögerung von bis zu einer Minute kommen, während webtrees Ihre Datenbank und Tabellen erstellt.


Login und erste webtrees Benutzung

Die Installation ist nun abgeschlossen. Sie sollten sich mit dem Benutzernamen und Passwort, das Sie im vorherigen Schritt gewählt haben, anmelden.


Einen Stammbaum erstellen

Bevor Sie auf irgendwelche Genealogie-Funktionen zugreifen können, müssen Sie einen Stammbaum erstellen. Der Name, den Sie hier wählen, wird Teil der Internet-Adresse Ihrer Website sein. Nachdem Sie einen Stammbaum erstellt haben, werden Sie in der Lage sein, Daten einzugeben oder eine GEDCOM-Datei hochzuladen.

Alternativ können Sie Ihre Stammbäume und Konfigurationseinstellungen aus einer vorhandenen PhpGedView-Installation auf dem gleichen Server importieren, indem Sie das PGV Import-Werkzeug benutzen.


Weitere Schritte

Es gibt viele Konfigurationsmöglichkeiten. Sie werden wahrscheinlich die Datenschutz-Einstellungen zuerst überprüfen wollen. Machen Sie sich nicht zu viel Gedanken über all die anderen Optionen - die Standard-Vorgaben sind für die meisten Fälle gut. Wenn Sie nicht weiterkommen, gibt es viel eingebaute Hilfe und Sie können freundliche Hilfestellung aus dem Hilfe Forum bekommen.
