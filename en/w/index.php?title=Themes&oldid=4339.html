In webtrees, a "Theme" controls the style or appearance of a web page - colour, font, icon size, and the like - as distinct from the content, the actual information being presented.

Adding, Creating and Editing Themes

Themes CANNOT be added, created or edited from within webtrees. This must be done at the coding level, by modifying existing Theme files, or adding new files and incorporating them into the webtrees package.

A number of Themes have been developed and are available for downloading via the webtrees.net website, at http://www.webtrees.net/index.php/en/add-ons/viewcategory/3

Built-In Themes

Each version of webtrees comes with a group of built-in Themes.

When first installed, the starting (default) theme is the webtrees theme. If they wish, users are able to select some other theme from the drop-down list that appears when they hover over the Themes menu on the top of the page.

Controlling Access To Themes

The Administrator can determine the default Theme and whether other Themes may be selected by users.

Default Theme For The Site

To set the default Theme for the site go to the Administration page and select Administration/Site Configuration/Site Configuration tab/

then Default Theme/click to edit choose from the dropdown list of possible themes.

Default Theme for a Specific Family Tree

To set the default Theme for a specific family tree go to the Administration page and select Family trees/{family tree title}/General tab/User Options/

then Default Theme choose from the dropdown list of possible themes.

Allow Users To Select Their Own Theme

To allow users to select their own Theme go to the Administration page and select Administration/Site Configuration/Site Configuration tab/

then Allow users to select their own theme click on the current setting and select from the dropdown list.

Show/Hide the Themes Menu Item for a Specific Family Tree

When first installed webtrees shows the Themes menu to all users, allowing them to select from the list of built-in Themes.

To show/Hide the Themes menu item for a specific family tree go to the Administration page and select Family trees/{family tree title}/General tab/User Options/

then Theme dropdown selector for theme changes select either the Ohide button or the Oshow button.
