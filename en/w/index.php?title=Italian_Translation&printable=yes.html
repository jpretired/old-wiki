Introduction

This page tries to offer a common place to all the people interested in translating webtrees into Italian.

It contains general translation guidelines, and the translation of some English terms that should be kept among all the translated phrases.

General translation guidelines

It's important that any translator read the Italian rules for good translations of the Free Translation Project - Sezione Italiana.
This document should be considered a reference document that should always be kept in mind.

In particular, you should adopt these rules:

  • You should avoid to italianize foreign terms (for example settare for setting). There are very few exception for terms that are very commonly used (for example cliccare for to click)
  • English texts typically addresses directly to the reader (Select another option). In Italian you should avoid it and use impersonal forms (Selezionare un'altra opzione).
  • English texts tend to be more polite that the Italian ones, using many please for the actions that the user should perform. Italian programs are more impersonal and simply tell what the user should do (Please select another option should be translated as Selezionare un'altra opzione).
