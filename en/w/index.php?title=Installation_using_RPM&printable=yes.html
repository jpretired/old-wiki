Installation using RPM

Linux RPM packages for webtrees on SUSE and openSUSE distributions are automatically built by the webtrees project on OpenSUSE Build Service [1].

For other Linux distributions, you can make your own branch on openSUSE Build Service.

Installation

The existing project packages can be installed in many way. For multiple examples, follow the "Download package link" at the webtrees project on OpenSUSE Build Service [2].

Example: Installing on SUSE Linux Enterprise Server 11 using the zypper command line tool

zypper addrepo http://download.opensuse.org/repositories/server:php:applications/SLE_11/server:php:applications.repo
zypper refresh
zypper install webtrees

Configuration

Installing this Linux RPM package will automatically resolve and install dependencies (such as MySQL and various php dependencies).

However there are still some additional configuration steps that must be taken before a webtrees site is up and running:

  • web address alias
  • set the timezone for PHP
  • create webtrees MySQL database
  • email

Web address alias

A URL such as http://www.yoursite.net/webtrees/ must be mapped to the webtrees files installed at /srv/www/webtrees/static/ by this RPM package.

By default Apache 2 is installed on SUSE and openSUSE Linux. To configure the above mapping with Apache, you can add the following line to an Apache configuration file (e.g. create a /etc/apache/conf.d/myconfig.conf): 

Alias /webtrees "/srv/www/webtrees/static"

Set PHP timezone

Create a file (for example 'timezone.ini') in /etc/php5/conf.d/ with the following contents: 

[Date]
; Defines the default timezone used by the date functions
; http://php.net/date.timezone
date.timezone = "UTC"
