In webtrees there are so to say two "front pages" for different purposes which only should be referenced by their respective names:

  • First is the Home page. This is the page most visitors see when they visit your site (unless in your configuration under Administration -> MyFamilyTree -> Privacy "Require visitor authentication" is set to "yes"). The content of this page can be varied for each family tree on your site.
  • Second is named My page and is a unique page for each of your registered users. When a user logs-in to your site from the "Home page" they will be taken to their own "My page". (Note: if they login from any other part of the site they will be returned to the page they selected "Login" from, once authenticated by the system). This page remains the same for each user and for any family tree they are allowed access to on your site.

Both of these pages consist of a number of "blocks" displaying different information about your family tree. These blocks can be moved (up, down, left, right) on the page or removed. There are different blocks that can be added to both pages. But some blocks are unique to one or the other page. Such an example are the blocks "favorites", which are named indentical on both pages, but could contain entirely different contents.

Configure "home page" blocks

There are two places where "home page" blocks can be configured, but in both cases only registered users with the role of "manager" or "administrator" can change them.

  • On the Administration pages you can set the default blocks that will be used for each new family tree that is added to your site (Administration -> Family trees -> Set the default blocks). This "default" setting only applies to trees added AFTER the default has been set. When you first install webtrees it is set to display blocks the developers have found from experience to be the most commonly used.
  • On the Home page itself, near the bottom of the page is a link titled "Change the blocks on this page". From that you can change the blocks for the specific family tree that you are viewing the Home page for.

Configure "My page" blocks

There are also two places where these blocks can be configured. In this case one is available only to administrators. That is on the page Administration -> Users Set the default blocks. This "default" setting only applies to new users added AFTER the default has been set.

Each user then has the ability to change the blocks on their own "My page". The link for this is at the bottom of their "My page", titled "Change the blocks on this page".

Regardless of the page you are configuring blocks for, or which link you use to edit the settings, the user-interface for changing these blocks will always look similar to the image below, changing only in color depending on the theme your site uses.

Link to change blocks

The options available are all straightforward, and require just two operations:

  1. Click on the block you want to add, remove or move
  2. Click on the appropriate arrows to move the block left, right, up, or down. The double left and right arrows move blocks from the main section to the right section or vice versa. Single left or right arrows move blocks to or from the "Available blocks" list, adding or removing them from either of the two display sections.
