Adding historical facts to your family tree

webtrees allows you to add dated facts to better situate the displays of your ancestors' lives in a historical context. This timeline of events is added by creating a list of data which will enable webtrees to display these historical events on the "Facts and events" tab of every individual's page. This function is only applicable to the individual page.

Configuring your website to display historical data

Adding the historical facts function to webtrees is easy. Create a file of events and dates and install it within the /data folder of your webtrees site. The file should be of standard php construction and titled in a manner to associate each sets of events with a language: "histo.xx.php" (e.g.: histo.fr.php for France or histo.en-US.php for US English). See Below. Replace the language code with the code of another country to add and events file pertaining to other countries. The installation of the histo.xx.php file will automatically give webtrees the ability to display this option. Visitors to your site will then simply tick the checkbox to show the "historical facts" which are related to the period of time during which each individual lived.

Creating a file histo.xx.php

Such a file is a php file which should respect a few simple presentation rules: Start with a header, respect the php coding rules, and then finally close the php code.
There is no need to understand php coding to create this file.
Use a text editor which allows for saving files without a BOM. Notepad++ which is an open source program (GPL license) may be downloaded at is such an editor, as is Text Wrangler for Mac users.

The file header

Cut and paste the following code: 

<?php
// File: /data/histo.xx.php (where xx is a language code such as "fr" or "en_GB")
// Display historical facts on the personal facts tab of the individual page 
// Each line is a GEDCOM record to describe an event.
// Events must contain dates, and will only be shown if the individual was alive on that date.

File content: the list of events

Here is an example:

Insert the following line (each line corresponds to one event): 

 $histo[] = "1 EVEN\n2 TYPE Front populaire\n2 DATE 03 MAY 1936\n2 NOTE Victoire du Peuple 
 (http://fr.wikipedia.org/wiki/Front_populaire_%28France%29)";

which will be displayed:

Note: This example shows that it is also possible to add a hypertext link. It would be correctly interpreted by webtrees.

What you could change is showned in red below: 

$histo[] = "1 EVEN\n2 TYPE Title\n2 DATE DD MMM YYYY\n2 NOTE Content of the note";
  • Title: is the title of the event. For example, type in Président de la République française.
  • DD MMM YYYY: is the date DD for day with one or two digits, MMM for month with three letters and YYYY for the year with four digits, (e.g. 8 JAN 1959).
  • Content of the note: write a descriptive text. For example, write Charles de Gaulle.

You would get the line: 

$histo[] = "1 EVEN\n2 TYPE Président de la République Française\n2 DATE 8 JAN 1959\n2 NOTE Charles de Gaulle";


Save the file.

Using the created file

Now, all you need to do is to add the file to the /data folder of webtrees and then, visit your family tree. Select an individual page and tick the checkbox Historical facts. You would see the events inserted within the facts and events of that individual.

Note: To allow letters with accents to be displayed correctly, the file should be saved with UTF-8 encoding. For historic reasons, webtrees will accept HTML entities such as &agrave; for à, however this is not recommended.

Some other possible data

It could also be interesting to add to the Historical facts:

  • significant scientific events for a family of scientists,
  • the major literary events for a family of writers,
  • important dates in history of art for a family of painters and artists,
  • etc...
