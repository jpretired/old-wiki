Introduction

Localizing the month names is not a 1:1 from English to other languages. In particular, some language use declensions for the nouns, so a name can change depending on the contest. This page aims to summarize the declensions related to the GEDCOM date specifications.


Declensions by language

Here's a list of languages that use declensions. It has been defined intersecting the languages reported in this forum message and the languages currently supported by webtrees.

Please note that for the languages we don't already know about, you'll see the name of the language itself instead of the declension name. This has been done to help maintain this page.
