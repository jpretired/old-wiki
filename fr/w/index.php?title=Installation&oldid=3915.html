L’installation de webtrees est vraiment facile. Vous n’avez besoin que d’un serveur web avec PHP et MySQL. Presque tous les services d’hébergement Web sont fournis dans le "package".

  • Télécharger webtrees-1.7.14.zip
  • Décompressez les fichiers et téléchargez-les dans un répertoire vide de votre serveur web.
  • Ouvrez votre navigateur Web et tapez l’URL de votre site webtrees (par exemple, http://www.yourserver.fr/webtrees) dans la barre d’adresse.
  • L’assistant de configuration webtrees démarre automatiquement. Il va vous poser quelques questions sur votre serveur, puis générer un fichier de configuration pour vous. À la fin de l’installation, il vous sera demandé de télécharger le fichier de configuration sur le serveur web. Il s’agit d’un contrôle de sécurité afin de s’assurer que seul le propriétaire du site peut le configurer.

C’est tout !

Toutefois, avant de pouvoir utiliser webtrees, vous avez besoin d’un (ou plusieurs) GEDCOM (un GEDCOM est un arbre généalogique normalisé) des fichiers. Si vous avez fait votre recherche en utilisant un programme de bureau tels que Family Tree Maker, vous pouvez utiliser la fonction "Enregistrer sous GEDCOM" pour créer un fichier GEDCOM. Si vous partez de zéro, alors webtrees peut créer un fichier GEDCOM pour vous. Alternativement, vous pouvez importer des données directement à partir de PhpGedView.

Ainsi, après installation, vous serez redirigé vers la page d’administration du GEDCOM (arbre généalogique), où vous devrez sélectionner une des options suivantes.

  • Ajouter un nouveau GEDCOM - Le fichier est chargé à partir du serveur. Si le fichier GEDCOM est trop grand pour être téléchargé, il est possible de le copier dans le dossier webtrees/data, et, avec cette option, le charger à partir de là.
  • Charger un nouveau GEDCOM - Cette option permet d'envoyer un fichier à partir de l'ordinateur local.
  • Créer un nouveau GEDCOM vide - Cela permet de démarrer à partir de zéro.
  • Transférer, les données existantes de PGV, directement dans webtrees, en utilisant [l’assistant de transfert] PGV-vers-webtrees.

Il y a beaucoup d’options de configuration. Vous aurez probablement envie de modifier les paramètres de confidentialité en premier. Ne vous inquiétez pas trop de toutes les autres options - les valeurs par défaut sont bonnes pour la plupart des gens. Si vous êtes coincé, il y a beaucoup d’aide intégrée et vous pouvez obtenir des conseils amicaux sur le forum d'aide en français (en anglais : http://webtrees.net/forums/).
