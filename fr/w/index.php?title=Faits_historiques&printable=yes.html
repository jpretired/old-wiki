Ajouter des faits historiques à votre arbre généalogique

webtrees permet l'ajout de données datées pour mieux situer vos ancêtres dans un contexte historique. La chronologie d'une généalogie est ajoutée en créant une liste de données afin que webtrees affiche ces événements historiques sous l'onglet "Faits et événements" de chaque page individuelle. Cette fonction n'est applicable que sur la page individuelle.

Configurer le site pour afficher les données historiques

Ajouter la fonction Faits historiques à webtrees est facile. Créez un fichier des événements et des dates et installez-le dans le dossier /data de votre site webtrees. Le fichier doit être en langage php standard et être nommé de manière à associer chaque liste d'événements à une langue "histo.xx.php" (e.g.: histo.fr.php pour la langue française ou histo.en_US.php pour l'anglais américain). Remplacer le code de langue par le code d'une autre langue pour ajouter les évènements d'un autre pays. L'installation d'un fichier histo.xx.php amènera automatiquement webtrees à afficher cette option. Il suffira de cocher la case Faits historiques pour afficher les faits reliés à la période durant laquelle un individu a vécu.

Créer un fichier histo.xx.php

Un tel fichier est un fichier php qui doit donc respecter quelques règles simples de présentation : débuter par un en-tête, respecter les règles d'écriture en php et finalement fermer le code php. Mais il n'est nul besoin de connaître la programmation php pour réaliser ce fichier. Il peut être fabriqué avec un simple éditeur de texte (Conseil: utiliser Notepad++ qui est un logiciel open source (licence GPL) et est téléchargeable), et le logiciel Text Wrangler pour les utilisateurs de Mac.

L'en-tête du fichier

Copier-coller le code suivant : 

<?php
// Début du fichier histo.xx.php
// Installer ce fichier dans le dossier /data de votre site webtrees sous le nom histo.xx.php
// Fichier utilisé pour afficher les faits historiques sur une page individuelle 
// Chaque ligne est un enregistrement de style GEDCOM décrivant un événement, incluant le caractère de nouvelle ligne (\n)
// Fichier à renommer : histo.xx.php où xx est un code de langue
//
// $Id: histo.demo.php 4028 2008-10-07 15:00:38Z fisharebest $

if (!defined('WT_WEBTREES')) {
   header('HTTP/1.0 403 Forbidden');
   exit;
}

Le corps du fichier: la liste des évènements

Partons d'un exemple:

Insérez la ligne ci-dessous (chaque ligne correspond à un enregistrement. La structure d'une ligne est simple mais précise.): 

$histo[] = "1 EVEN\n2 TYPE Front populaire\n2 DATE 03 MAY 1936\n2 NOTE Victoire du Peuple 
 (http://fr.wikipedia.org/wiki/Front_populaire_%28France%29)";

qui sera affichée:

Note: Cet exemple a été choisi pour montrer qu'il est aussi possible d'utiliser un lien hypertexte. Il sera correctement interprété.

Ce que vous pouvez modifier dans la ligne est en rouge ci-dessous: 

$histo[] = "1 EVEN\n2 TYPE Titre\n2 DATE JJ MMM AAAA\n2 NOTE Contenu de la note";
  • Titre: c'est le titre de notre évènement. Par exemple inscrivez Président de la République française.
  • JJ MMM AAAA: c'est la date avec le jour sur un ou deux chiffres, le mois en 3 lettres (système international basé sur l'anglais) et l'année sur 4 chiffres.. Par exemple, inscrivez 8 JAN 1959.
  • Contenu de la note: rédigez un texte descriptif. Par exemple, inscrivez Charles de Gaulle.

Vous devriez obtenir la ligne: 

$histo[] = "1 EVEN\n2 TYPE Président de la République Française\n2 DATE 8 JAN 1959\n2 NOTE Charles de Gaulle";

La fin du fichier

Fermons proprement le code php. 

/* Fin du fichier histo.xx.php */ 
?>

Enregistrez votre fichier.

Utiliser le fichier créé

Il ne reste plus qu'à l'installer dans le répertoire /data de webtrees et à visiter votre arbre généalogique.

Pour vérifier, choisissez la fiche d'une personne née avant 1936 et décédée après 1959. Cochez la case Faits historiques et vous devriez voir les deux événements insérés parmi les événements de la vie de la personne.

Note: pour que les lettres accentuées s'affichent correctement, le fichier devrait être enregistré en encodage UTF-8. Pour des raisons historiques webtrees accepte les codes HTML (exemple : &agrave; pour à), toutefois cela n'est pas recommandé.

Codes des caractères accentués

Si vous avez la désagréable surprise de découvrir que les caractères accentués sont mal interprétés, et que vous ne pouvez pas convertir le fichier en UTF-8, il vous faut alors les encoder en HTML (liste exhaustive (en)) : 

à : &agrave;
â : &acirc;
ç : &ccedil;
è : &egrave;
é : &eacute;
ê : &ecirc;
ë : &euml;
î : &icirc;
ï : &iuml;
ô : &ocirc;
ù : &ugrave;
û : &ucirc;
ü : &uuml;
€ : &euro;

Autres événements possibles

Aux faits historiques importants, il pourrait être intéressant d'ajouter :

  • les événements scientifiques marquants pour une famille de scientifiques,
  • les grands événements littéraires pour une famille d'écrivains,
  • les dates importantes de l'histoire de l'art pour une famille de peintres ou d'artistes,
  • etc...
