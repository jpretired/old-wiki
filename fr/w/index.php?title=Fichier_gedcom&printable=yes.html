Fichier gedcom

De webtrees
Aller à : navigation, rechercher


Le fichier GEDCOM est un fichier texte normalisé qui représente l'arbre généalogique. Ce fichier permet des échanges et peut être issu d'un logiciel ou généré par webtrees. Plusieurs arbres peuvent être présent en même temps, sur webtrees. Sur la page "Administration GEDCOM" deux options sont offertes : travailler sur un des arbres généalogiques existants dans webtrees ou en introduire un nouveau.

Attention

1) Les options décrites ci-après, fonctionnent avec un fichier texte GEDCOM, avec l'extension .ged, mais sans compression.

2) Le fichier GEDCOM une fois introduit dans webtrees n'a plus d'utilité pour webtrees.

Travailler sur un arbre généalogique existant

  • Configurer - Cela permet de paramétrer, pour ce fichier GEDCOM, de nombreux éléments tels : le titre, la langue, le calendrier (Calendrier révolutionnaire), les données de contact, les données de référencement (balises META), etc., la confidentialité, les objets multimédias, les règles d'accès (visiteurs, utilisateurs), l'affichage (noms, diagrammes, pages), le masquage ou l'affichage (diagrammes, pages, etc.), l'édition.

Nota - Le paramétrage par défaut est satisfaisant et seul quelques paramètres sont, éventuellement, à retoucher.

  • Exporter - Les données généalogiques contenues dans la base de données sont exportées, sous forme d'un fichier texte (GEDCOM), dans data/NomDuFichierGEDCOM.ged.
  • Importer - Le fichier GEDCOM (xxxx.ged,) placé dans le répertoire data/, sera incorporé à la base de données.

La case "Si vous avez créé des objets médias dans webtrees, et avez modifié votre gedcom hors-ligne en utilisant un programme qui supprime les objets médias, cochez cette case pour fusionner les objets média en cours avec le nouveau GEDCOM", dans la majorité des cas, reste décochée.

  • Télécharger (Download) - Les données généalogiques de la base de données sont rapatriées, sur votre poste, sous forme d'un GEDCOM. Quelques options sont paramétrables comme : la compression, le codage (UTF8, ANSI), balises, média, séparateur.
  • Téléchargement - Le fichier GEDCOM est lu mais reste là où il est pris. Les données sont transférées dans la base de données. Le nom du fichier d'origine n'a pas d'importance.

La case "Si vous avez créé des objets médias dans webtrees, et avez modifié votre gedcom hors-ligne en utilisant un programme qui supprime les objets médias, cochez cette case pour fusionner les objets média en cours avec le nouveau GEDCOM", dans la majorité des cas, reste décochée.

  • Supprimer - Les données de ce GEDCOM sont supprimées de la base de données.

Introduire un nouveau GEDCOM

  • Fichier par Défaut - Lorsqu'un nouveau GEDCOM est introduit, cette liste déroulante permet de sélectionner l'arbre généalogique qui s'affichera par défaut.
  • Ajouter un nouveau GEDCOM - Le comportement est similaire à "Importer".
  • Charger un nouveau GEDCOM - Le comportement est similaire à "Téléchargement".
  • Créer un nouveau GEDCOM vide - Cela permet de démarrer un tout nouvel arbre généalogique par saisie directe.

Important

Le choix entre "Importer" et "Téléchargement" réciproquement "Ajouter un nouveau GEDCOM" et "Charger un nouveau GEDCOM" est lié à la taille du fichier. Dès que la taille du fichier GEDCOM dépasse 2 Mo, l'option "Téléchargement" n'est plus possible ; il faut choisir l'option "Importer". Avec l'option "Importer", le fichier GEDCOM, de grosse taille, doit être, auparavant, transféré par FTP dans le dossier /data. C'est là qu'il sera pris en compte.

Format

Plusieurs formats de codage du GEDCOM sont utilisés. webtrees reconnaît plusieurs formats (ANSI, MAC, DOS, pas encore ANSEL) mais le transfert sera plus lent. Il faut privilégier le format UTF8.
