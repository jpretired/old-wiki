Convertir depuis PhpGedView

Si vous vous déplacez vers webtrees à partir d'une configuration existante PhpGedView (PGV), et si votre installation PGV répond à certaines exigences, webtrees a prévu un «assistant» de transfert de la majorité de vos données en une opération relativement rapide et indolore (voir les exceptions ci-dessous). Veuillez noter que nous avons conçu cet assistant afin de ne pas perturber votre installation existante PGV, en laissant tous vos paramètres, vos données et votre site intact et pleinement fonctionnel.

Les exigences sont les suivantes :

  1. La base de données PGV et répertoire de lancement doivent être sur le même serveur que webtrees ;
  2. votre nom d'utilisateur de la base de données MySQL de webtrees et le mot de passe associé doivent être les mêmes que votre nom d'utilisateur et votre mot de passe PGV, ou, si vous avez créé un nouvel utilisateur pour webtrees, cet utilisateur doit avoir tous les privilèges pour accéder à votre base de données PGV ;
  3. la version PGV doit être la 4.2.3. ou la 4.2.4. (ce qui correspond à une opération interne "PGV_SCHEMA_VERSION" entre 10 et 14) ; les versions plus récentes, y compris la version SVN actuelle 4.3 et les versions SVN à venir, fonctionneront probablement, si les structures des données n'ont pas changé ;
  4. tous les changements dans PGV doivent être acceptés (sinon les modifications en attente ne seront pas transférées) ;
  5. tous les utilisateurs PGV existants doivent avoir une adresse de courriel, et cette adresse doit être unique pour chaque utilisateur (PGV permet aux utilisateurs de supprimer leur adresse de courriel, ou d’avoir la même adresse que d’autres utilisateurs alors que webtrees exige que chaque utilisateur ait sa propre adresse unique de courriel);
  6. l'assistant du processus de transfert remplace le nom d'utilisateur et le mot de passe entrés lors de la création du compte initial sous webtrees. Le nom de l'administrateur principal et le mot de passe seront alors identiques au nom d'utilisateur et mot de passe de l'administrateur de PGV. Il reste, bien sûr, possible de gérer le compte et donc de modifier le mot de passe.

Remarque importante : Cet assistant de transfert n'est pas en mesure d'aider au déplacement des éléments multimédias. Vous aurez besoin de mettre en place et de déplacer ou copier votre configuration des médias et tous les objets médias séparément quand l'assistant de transfert aura terminé. Si vous utilisez le pare-feu des médias dans PGV et un répertoire en dehors de la PGV, il faudra dupliquer la configuration des médias dans le répertoire médias de webtrees, et le répertoire de configuration du pare-feu des médias de PGV sera à la disposition de webtrees.

Quand le transfert sera terminé, vous devriez vérifier la configuration de votre arbre généalogique ainsi que les paramètres de confidentialité. En raison de différences dans les formats de données internes, les paramètres suivants ne sont pas transférés pour l’instant : les restrictions personnalisées de confidentialité, les paramétrages des blocs, la FAQ et les blocs HTML. Nous espérons les ajouter à l'assistant dans une prochaine version.

L'assistant de transfert est accessible dans webtrees depuis la page « Administration GEDCOM », page à laquelle vous serez automatiquement redirigé à la fin des étapes d'installation de webtrees (page à lire : Installation). Cette option n'est disponible que sur une nouvelle installation vide de webtrees ; dès que vous aurez créé un fichier GEDCOM (un arbre généalogique) ou dès que vous aurez ajouté des comptes d'utilisateurs, cette fonction ne sera plus disponible.
