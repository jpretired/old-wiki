Backup

De webtrees
Aller à : navigation, rechercher

Les sauvegardes sont utiles voir même indispensables. Quel que soit le problème que vous rencontriez, il peut presque toujours être récupéré à partir d’une bonne sauvegarde.

Pour faire une sauvegarde de webtrees, vous avez besoin de faire une copie des documents suivants

  1. Les fichiers du dossier webtrees/data.
  2. Les tables de la base de données. Des outils sont disponibles gratuitement, comme PhpMyAdmin, outils qui permettent de sauvegarder toutes les tables en un seul clic.

Note de Jean-Jacques - Une précaution supplémentaire est de sauvegarder la dernière version du GEDCOM. Cela permet, de sauvegarder vos données généalogiques déjà incluses dans la base de données, mais ainsi, éventuellement et surtout, de repartir à zéro.

Personnalisation

  • Si vous avez créé des fichiers de traductions personnalisées (webtrees/data/language), vous devez également les sauvegarder.
  • De même pour vos propres modifications du modèle/template, des rapports, etc.

Rappelez-vous que pour la plupart des services d’hébergement Web, la sauvegarde de vos données est de votre responsabilité.
