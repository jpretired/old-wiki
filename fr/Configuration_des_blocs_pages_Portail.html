Configuration des blocs pages Portail

De webtrees
Aller à : navigation, rechercher

Il y a deux pages "Portail" dans webtrees:

  • La première est la page "Accueil". C'est la page que la plupart des visiteurs voient lorsqu'ils visitent votre site pour la première fois (à moins que l'accès soit toujours configuré à "Authentication obligatoire"). Le contenu de cette page peut varier pour chaque arbre généalogique de votre site.
  • La deuxième est nommée "Ma page" et elle est unique pour chacun de vos utilisateurs. Lorsqu'un utilisateur se connecte à votre site à partir de la page "Accueil", il est redirigé automatiquement sur sa page "Ma page". (Note: s'il se connecte à partir de n'importe quelle autre page du site, il restera sur ladite page une fois connecté.). Cette page demeure la même pour tous les arbres généalogiques auxquels cet utilisateur a accès.

Ces deux pages consistent en un nombre de blocs affichant différentes informations de l'arbre généalogique. Ces blocs peuvent être déplacés (haut, bas, gauche, droite) dans la page ou être enlevés. Il y a aussi différents blocs qui peuvent être ajoutés à chacune des pages, certains sont communs aux deux pages, et d'autres sont uniques à l'une ou l'autre page.

Configurer les blocs de la page "Accueil"

Il y a deux endroits où vous pouvez configurer les blocs de la page Accueil. Mais dans les deux cas, seuls les utilisateurs ayant un rôle de "gestionnaire" ou "administrateur" peuvent les modifier.

  • Sur les pages Administration, vous pouvez définir les blocs par défaut qui seront utilisés pour chaque nouvel arbre généalogique ajouté à votre site (Administration -> Arbres généalogiques -> Définir les blocs par défaut). Ce paramètre "Définir les blocs par défaut" s'applique uniquement aux arbres ajoutés APRÈS que le défaut a été défini. Lorsque vous installez webtrees pour la première fois, il est configuré pour afficher des blocs définis (déduits par expérience) par les développeurs comme étant les plus couramment utilisés.
  • Sur la page Accueil, au bas de la page, il y a un lien titré "Changer les blocs de cette page". Cliquer sur ce lien vous permet de modifier les blocs de la page Accueil de l'arbre généalogique sélectionné. Vous pouvez le faire pour chaque arbre généalogique de votre site.

Configurer les blocs de la page "Ma page"

Il y a aussi deux endroits où vous pouvez configurer ces blocs. Le premier est accessible seulement aux administrateurs. Il est situé sur la page Administration -> Utilisateurs -> Définir les blocs par défaut. Ce paramètre "Définir les blocs par défaut" ne s'applique qu'aux utilisateurs ajoutés APRÈS que le défaut a été mis en place.

Chaque utilisateur a alors la possibilité de modifier les blocs de sa propre page "Ma page". Le lien de cette option est situé au bas de sa page "Ma page", titré "Changer les blocs de cette page".

Link to change blocks

Quelle que soit la page dont vous configurez les blocs ou quel que soit le lien que vous utilisez pour modifier les paramètres, l'interface de modification de ces blocs ressemblera à l'image ci-dessous, sauf pouvant être de couleur différente selon le thème utilisé par votre site.

Block configuration

Les options disponibles sont simples et nécessitent seulement deux opérations :

  1. Cliquer sur le bloc que vous voulez ajouter, enlever ou déplacer
  2. Cliquer sur les flèches appropriées pour déplacer le bloc: gauche, droite, haut ou bas. Les flèches doubles gauche et droite déplacent les blocs de la section centrale vers la section de droite ou vice versa. Les flèches simples gauche ou droite déplacent les blocs de, ou vers, la liste "Blocs disponibles", les ajoutant ou les enlevant de l'une ou de l'autre des sections.
